bloque 3: operadores

¿qué son los operadores?

Los operadores son aquellos símbolos que le indican a Python que realice alguna operación específica, ya sea aritmética, lógica, comparativa, etc.
En Python podemos encontrar los siguientes tipos de operadores:

  • Operadores aritméticos
  • Operadores relacionales
  • Operadores bit a bit
  • Operadores de asignación
  • Operadores lógicos
  • Operadores de pertenencia
  • Operadores de identidad

Como puedes ver hay una buena variedad de estos, pero de momento solo procederemos a los mas importantes o de los cuales vamos a hacer un uso constante a lo largo de este curso („• ᴗ •„).

operadores aritméticos

Pues como su nombre lo indica son aquellos operadores que nos permiten realizar operaciones aritméticas, recibiendo dos operando como entrada y devuelve un resultado. Por ejemplo: suma = 5 + 7, siendo que 5 y 7 son los operandos y el + el operador aritmético y como notaras el resultado será almacenado en la variable suma.
Procedamos a ver una lista de los operandores aritméticos que podemos encontrar en Python:

operador descripción ejemplo
+ realiza la suma de operandos 12 + 4 = 16
- realiza la resta entre operandos 23 - 4 = 19
* realiza la multiplicación entre operandos 12 * 2 = 24
/ realiza la división entre operandos 24 / 2 = 12
% devuelve el módulo entre operandos 16 % 3 = 1
** realiza la potencia entre operandos 3 ** 2 = 9
// realiza la división dando el resultado entero 17 // 7 = 2

algo que debes saber...

El sistema PEMDAS se aplica mucho en la programación por lo que debemos tener cuidado al momento de realizar nuestras operaciones, recuerda que lo que esté dentro de un () tendrá más prioridad al momento de realizarse las operaciones, por ejemplo no es lo mismo poner 3+3+12*3/2 a poner 3+3+(13*3)/2, así que recuerda, paréntesis > exponente > multiplicación y división > suma y resta.

operadores relacionales

Los operadores relacionales los usamos cuando queremos comparar y/o establecer relaciones entre datos, internamente Python devuelve un True o un False dependiendo de el resultado de la comparativa.
Procedamos a ver una lista de los operadores relacionales que podemos encontrar en Python:

operador descripción ejemplo
> Retorna True si el valor de la derecha es menor 12 > 10 = True
< Retorna True si el valor de la izquierda es menor 5 < 10 = True
== Retorna True si ambos valores son iguales 7 == 5 = False
>= Retorna True si el valor de la derecha es menor o igual 10 >= 10 = True
<= Retorna True si el valor de la izquierda es menor o igual 12 <= 10 = False
!= Retorna True si ambos valores son distintos 23 != 12 = True

Cabe recalcar que dichos operadores se pueden utilizar de igual manera con los strings, pero el comportamiento de estos es un poco distinto, pero no te preocupes profundizaremos en eso cuando lleguemos al bloque de strings (⌒‿⌒).

operadores de asignación

Como su nombre lo indica son operadores que utilizamos para dar valores a las variables o modificar los valores que éstas almacenen, recordemos cómo asignamos valores a las variables: a = 3, aquí el signo = está fungiendo como el operador de asignación. Pero también tenemos cosas cómo:
a += 10, esto es equivalente a que digamos a = a + 10.

por si no lo sabes...

Lo que realizamos en el ejemplo de a = a + 10 fue una reasignación de variable donde el valor que posee originalmente a ahora pasará a ser ese mismo sumado con 10. Por ejemplo tenemos la siguiente variable x = 3, si queremos que x se multiplique por 3 y este resultado se almacene en la misma x solo debemos decir
x = x * 3 y ahora x = 9.

Procedamos a ver una lista de los operadores de asignación que podemos encontrar en Python:

operador descripción y/o equivalencia
= x = 7, el valor de 7 se asigna a x
+= x += 7 es equivalente a x = x + 7
-= x -= 7 es equivalente a x = x - 7
*= x *= 7 es equivalente a x = x * 7
/= x /= 7 es equivalente a x = x / 7
%= x %= 7 es equivalente a x = x % 7
**= x **= 7 es equivalente a x = x ** 7
//= x //= 7 es equivalente a x = x // 7

operadores lógicos

Los operadores lógicos nos sirven en ciertos casos similares a lo que realizamos con los comparativos, pero no te preocupes si te parece confuso los veremos más en acción a medida que vayamos avanzando en el curso (¬‿¬ ). En Python se representan de la siguiente manera:

operador descripción ejemplo
and Devuelve True si ambos operandos son True a and b
or Devuelve True si uno de los operandos es True a or b
not Devuelve True si uno de los operandos es False not a
