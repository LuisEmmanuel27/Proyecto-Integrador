bloque 1: primeros pasos en python

¿que es python?

Desarrollado a fines de la década de 1980 por Guido van Rossum en el Instituto Nacional de Investigación de Matemáticas y Ciencias de la Computación en los Países Bajos. La versión inicial fue publicada en el grupo de noticias alt.sources en 1991, y la versión 1.0 fue lanzada en 1994.

Es un lenguaje de programación, multiparadigma y multinivel, con soporte en programación orientada a objetos, imperativa y funcional. Con este tipo de lenguaje se pueden crear aplicaciones nativas e híbridas, y cuenta con una sintaxis accesible para las personas con un nivel de 'alfabetización' básico en lenguajes de programación.

usos y aplicaciones de python

Python es una tecnología imprescindible en plataformas de entretenimiento y social media, destinado al aprendizaje automático y a la construcción de algoritmos de recomendación, por eso aplicaciones como Instagram, Pinterest, Dropbox, Facebook, Spotify y Netflix tienen este lenguaje en su desarrollo porque permite programar tareas con el objetivo de procesar grandes cantidades datos y obtener información de valor.
Además, es posible abordar los siguientes tipos de proyectos con el lenguaje:

  • aplicaciones web
  • ciencias de datos
  • aprendizaje automático o machine learning
  • análisis y automatización de datos
  • Inteligencia artificial

ventajas de python

Python es un lenguaje con múltiples ventajas las cuales unas de las tantas son las siguientes:

  • Resulta un buen lenguaje para aquellos que deseen iniciarse en el mundo de la programación, principalmente por sus múltiples campos de aplicación.
  • Frameworks y entornos integrados para el desarrollo ágil y eficiente de aplicaciones web.
  • Se puede integrar con los lenguajes C, C ++, COM, ActiveX, CORBA y Java.
  • Puede utilizarse programación orientada a objetos, estructurada o funcional.
  • Es interpretado y no compilado, siendo el debugging (la depuración) más veloz.

Y si con todo lo anterior aún no te parece una buena opción para comenzar a aprenderlo, tenemos aquí una pequeña lista de razones del por que Python es un buen lenguaje para empezar:

  • Hace énfasis en la legibilidad y claridad del código porque es sencillo escribir determinadas instrucciones o realizar procesos especificos con el lenguaje.
  • Es multiplataforma, esto quiere decir que podrás crear aplicaciones en el sistema operativo que te toque trabajar: Windows, Linux, IOS, siendo este muy portable.
  • Al funcionar Python a través de un intérprete lo convierte en un lenguaje de fácil traslado, ya que podrás ejecutar programas en diversas plataformas.
  • Si estás interesado en la creación de interfaces gráficas de una forma sencilla, Python también es la opción recomendada porque cuenta con una biblioteca predefinida que te permitirán elaborar botones, controles de lista, cuadros, tablas y todo lo necesario para tener una UI dinámica.

Entre muchas otras ventajas, así que ¿qué esperas? ¡Comencemos a aprender Python! (/≧▽≦)/.

¿dónde comenzamos?

A estas alturas te estaras preguntando, "Ya pero ¿Donde programo con Python? (⊙_⊙)" Pues hay múltiples medios para ello, de los cuales Pymu recomienda los siguientes 5 (da click en los iconos para saber más y el como instalar):

1. google colab

Colab es el medio que nos ofrece google para programar Python en estilo notebook, donde no es necesario la instalación en nuestras computadoras, solo una cuenta de google y conexión a internet es lo que requerimos para su uso.

2. jupyter notebook

Con Jupyter Notebook lo que tenemos es algo similar a colab, solo que este si debe ser descargado e instalado en nuestras computadoras para su funcionamiento, podemos instalarlo siguiendo los pasos indicados en su pagina web o con anaconda.

3. pycharm

El autodenominado como la mejor IDE para programar Python, PyCharm proporciona una finalización del código inteligente, inspecciones del código, indicación de errores sobre la marcha y arreglos rápidos, así como refactorización de código automática y completas funcionalidades de navegación.
Para los novatos en la programación esta puede ser una de las mejores opciones ya que como se mencionó antes, PyCharm es de código inteligente por lo que ayuda mucho a reducir los errores que se puedan cometes al momento de codificar, pero también es el más demandante en los recursos de la computadora.

4. sublime text

Es uno de los más populares en general, Sublime Text es un editor de Texto para escribir código en casi cualquier formato de archivo. Está especialmente pensado para escribir sin distracciones, esto quiere decir que visualmente ofrece un entorno oscuro donde las líneas de código que escribas resaltarán para que puedas centrarte exclusivamente en ellas, una de sus principales ventajas es que es muy ligero por lo que consume pocos recursos, aunque su configuración no es del todo fácil para los novatos y otro inconveniente es que es de pago.

5. visual studio code

Visual Studio Code es el editor desarrollado por Microsoft, es compatible con varios lenguajes de programación y un conjunto de características que pueden o no estar disponibles para un lenguaje dado.
Una de las mayores fortalezas de este editor es que contiene múltiples extensiones para ayudarnos en la codificación haciendolo tambien un editor inteligente como PyCharm, también muy recomendado si vas empezando por el mundo de la programación, eso si deberás investigar bien cuales son las mejores extensiones para lo que buscas trabajar.
