bloque 6: sentencia condicional if

¿qué es un if?

En caso de que no sepas que es un if, pues es algo bastante sencillo pero muy muuuuy util en el mundo de la programación, los if son usados para definir una sentencia, un especie de filtro por asi decirlo donde si se cumple cierta condición se realiza cierta acción, en los if haremos uso principalmente de los operadores relacionales y en determinadas ocasiones de los lógicos ( •̀ ω •́ )✧.

¿cómo se usa en Python?

La estructura de un if es bastante simple, como mencione antes en la definición es un condicional donde si se cumple se realiza cierta acción, pues entonces veamos un ejemplo simple de esto:

Como se ve en el ejemplo tenemos dos variables con sus respectivos valores X y Y, seguido se coloca el if y los paréntesis que contendrán la condición a cumplir, seguido se colocan dos puntos : esto para indicar que estamos abriendo el if, en otros lenguajes se colocan {}, pero en Python no es el caso, si las colocamos nos mandara un error, notemos que el condicional que se colocó es que X debe ser mayor que Y, como es el caso se ejecuta el contenido del if que en este caso es el print, de no cumplirse la condición simplemente Python no haría nada (ㆆ_ㆆ).

if y else

Puede que te preguntes "Sí bueno tia Pymu entonces, ¿podemos hacer algo en caso de que no se cumpla la condición?" y la respuesta es si, lo que podemos hacer es utilizar un else, los else sirven para hacer una acción en caso de que NO se cumpla la condición, veamos un ejemplo para que esto nos quede más claro:

Podrás notar que ahora la condición es que X sea menor que Y lo cual no es cierto, por lo que Python procede a pasar al else del cual ejecuta el print como se ve en el ejemplo.

Algo importante MUY importante que debes tener en cuenta es que la identación (el espacio que se coloca cuando estamos dentro de un condicional) es realmente clave, si está mal colocada pueden cometerse múltiples errores, siempre ten cuidado al respecto, ya que en este caso si quitamos la identación estamos indicando que estamos fuera del if, mejor veamos el siguiente ejemplo para que veas a que me refiero:

algo curioso...

En Python puedes prescindir de usar los () siendo que puedes escribir un if de la siguiente manera if x < y:. No lo recomiendo a menos que estés totalmente seguro o segura de que no entraras en confusión（￣︶￣）.

más if, más else y la pyramid doom

Algo que podemos hacer es una cadena de else if, cuando ocupamos más condiciones que queremos colocar por medio de un if, para ello se hace uso de los elif, miremos un ejemplo para que entiendas como funciona:

Como puedes notar en el ejemplo con elif se colocó otra condición que bien puede o no cumplirse, en el caso del ejemplo puedes notar que no se cumple por lo que se pasa a el else que coloca el ultimo print, esto es bastante útil en determinados casos PERO esto también puede ser problemático (⊙_⊙;). En caso de que requieras hacer múltiples condiciones tendrías que escribir una absurda cantidad de elif, lo cual no es precisamente eficiente o considerado como una buena practica, lo que da origen a la temible pyramid doom o piramide de la fatalidad (⊙ｏ⊙), que bueno, en realidad en Python más que una pirámide es una torre de elif, pero no te preocupes en bloques futuros veremos otras estructuras que nos ayudarán a evitar esto (╹ڡ╹ ).
