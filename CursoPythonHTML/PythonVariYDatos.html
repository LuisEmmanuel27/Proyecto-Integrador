bloque 2: variables y tipos de datos

definición de variables

Puede que ya tengas conocimientos previos acerca de de las variables sobre qué son y el cómo funcionan, pero en caso de que no, igualmente vamos a definirla (╹ڡ╹). Una variable es un nombre que se refiere a un objeto que reside en la memoria. El objeto puede ser de alguno de los tipos de datos que existen en Python los cuales veremos más adelante. Imagina que las variables son cajas que almacenan información.

Cada caja tiene su nombre o identificador único aunque claro, el contenido de dichas cajas puede ser modificado según las tareas que estemos realizando.
En las variables encontramos lo que denominamos el alcance de las variables, osea si dicha variable se usará en cualquier sitio del código se les conoce como globales o solo en bloques específicos del mismo las cuales se conocen como locales.
Pero bueno, no pasa nada si aún no te queda del todo claro sobre a qué nos referimos con variables locales y globales, más adelante profundizaremos en dicho tema.

tipos de datos

En Python podemos encontrar distintos tipos de datos que tienen distintas características y clasificaciones, los más básicos serían los de tipo numérico, entero y booleano. Que si bien, como mencionamos antes, en Python no es necesario asignar qué tipo de valor tendrán nuestras variables, si debemos saber de qué tipo se tratan ya que entre estas hay diferencias significativas como veremos a continuación (¬‿¬).

1. int

Las variables de tipo entero por lo general siempre se les conoce como int, estos son basicamente números fijos o precisos, para que me entiendas los que no tienen decimal (￣▽￣)".

2. long

Similar int contamos con el tipo de dato long que en esencia es un int pero con mucho mayor rango, siendo que este está limitado sólo por la memoria de nuestra máquina (⊙o⊙), aunque en este caso, sí que debemos indicar que estamos usando un long colocando una L de la siguiente manera: numeroLong = 33L;
